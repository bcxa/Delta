Delta

A set of libraries and tools, all sharing the Ms-RL license.

  • Delta.Wsq: .NET WSQ (Image format primarily used to store fingerprints) encoder/decoder based on the NIST implementation. Provides also a Paint.NET plugin.
  • Delta.CapiNet: .NET around Microsoft's crypt32.dll.
  • Delta.CertXplorer: GUI allowing to explore the machine's certificate store as well as decoding any ASN.1 encoded file.
  • Delta.ImageRenameTool: A simple GUI allowing to rename multiple images at one time.
